Pseudo Elementos

Usados para adicionar elementos extras na página, sem a necessidade de declará-lo no HTML, ou para selecionar uma parte de algum elemento.

Identificado por :: palavra-chave

Pseudo Elemento :: first-letter

Seleciona a primeira letra da primeira linha

Propriedades disponíveis:
  • color
  • background
  • margin
  • padding
  • border
  • font
  • Propriedades de Texto

Pseudo Elemento :: first-line

Seleciona a primeira linha de um texto

Propriedades disponíveis:
  • color
  • background
  • font
  • Propriedades de Texto

Pseudo Elemento :: backdrop

Estiliza o plano de fundo de elementos em modo de fullscreen como dialog ou video

Pseudo Elemento :: marker

Permite estilizar os tópicos ou números de uma lista (li) ou então a flecha de um elemento summary

Propriedades disponíveis:
  • color
  • content
  • white-space
  • font
  • animation
  • transition

Pseudo Elemento :: place-holder

Um placeholder é uma dica para ajudar no preenchimento de algum campo de formulário

Propriedades disponíveis:
  • color
  • background
  • font
  • text

Pseudo Elemento :: selection

Permite estilizar a seleção de textos no navegador

Propriedades disponíveis:
  • color
  • background
  • text
