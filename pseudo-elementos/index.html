Pseudo Elementos

Usados para adicionar elementos extras na página, sem a necessidade de declará-lo no HTML, ou para selecionar uma parte de algum elemento.

Identificado por :: palavra-chave

Pseudo Elemento :: first-letter

Seleciona a primeira letra da primeira linha

Propriedades disponíveis:
  • color
  • background
  • margin
  • padding
  • border
  • font
  • Propriedades de Texto
